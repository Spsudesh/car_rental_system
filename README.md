# 🚗 Car Rental Management System (MERN)

A full-stack car rental application built using the MERN stack where users can browse cars and book rentals, while admins manage cars and bookings via dashboard.

---

## ✨ Features

- User Login / Signup  
- Browse & Book Cars  
- Admin Dashboard  
- Add / Manage Cars  
- Manage Bookings  
- Booking Status Control  

---

## 🛠 Tech Stack

- React.js  
- Node.js  
- Express.js  
- MongoDB  
- Tailwind CSS  
- JWT Authentication
